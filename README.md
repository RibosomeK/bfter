# Bfter

a simple [Brain Fuck](https://brainfuck.org/) interpreter in rust

# Usage

```
cargo run -- --help

Usage: bfter [FILE]

Arguments:
  [FILE]  The path of BrainFuck source file [default: ./sample/hello.bf]

Options:
  -h, --help  Print help
```

# Build

```
cargo build --release
```
